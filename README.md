# @orchestra-mcp/devtools

Developer tools components for Orchestra MCP — terminal, debugger

Part of the [Orchestra MCP](https://orchestra-mcp.dev) ecosystem.

## Install

```bash
npm install @orchestra-mcp/devtools
# or
pnpm add @orchestra-mcp/devtools
```

## Usage

```tsx
import { } from '@orchestra-mcp/devtools'
```

## Requirements

- React >= 18.0.0
